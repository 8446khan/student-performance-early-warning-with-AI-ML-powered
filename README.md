🤖 AI-Based Student Early Warning System
📌 Overview

This is a machine learning-powered web application that analyzes student performance data to identify at-risk students and provide early academic intervention. It helps institutions improve student success rates using predictive analytics.

🎯 Features
Student performance data analysis
ML-based risk prediction
Early warning system for at-risk students
Interactive dashboard
Data visualization for insights
Admin-friendly interface


🛠️ Technologies Used
Python 🐍
Flask (Backend)
Machine Learning (Scikit-learn / Pandas / NumPy)
HTML, CSS, JavaScript
React js
PostGresql
